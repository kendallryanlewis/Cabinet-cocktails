### Paywall Strategy

**Trigger Points**:
1. **Hitting Limits**: When user tries to add 21st ingredient to cabinet
2. **Feature Discovery**: When user taps on a premium feature
3. **Value Moments**: After user finds a great cocktail they want to save (11th favorite)
4. **Onboarding**: Soft introduction during tutorial (no hard sell)
5. **Success Moments**: After user successfully makes a cocktail
6. **App Launch**: Occasional gentle reminder (max once per week)

**Paywall Design**:
- Clear value proposition
- Feature comparison table (Free vs. Premium)
- Social proof (user testimonials)
- Money-back guarantee messaging
- No dark patterns - easy to dismiss
- Restore purchases button prominent
